About the GoCredi Model

GoCredi is a smart application that helps individuals determine the likelihood of their credit card application being approved based on their personal and financial data.

🔍 How does the model work?

The system relies on machine learning algorithms such as:

  • K-Nearest Neighbors
  • Random Forest
  • Voting Ensemble
These algorithms are trained using real credit card application data, and the most accurate model is selected to provide the best prediction.

⚙️ What do we need for prediction?

Once the data is entered into the model, the system analyzes it internally using the trained model and provides the result instantly.
