About this Project

This website is a student graduation project submitted as part of the "Programming Project" course, which is a core requirement in the Intelligent Systems Engineering and Computer Engineering programs at Palestine Ahliya University. The project was built using Machine Learning and web technologies to predict credit card approval likelihood.
